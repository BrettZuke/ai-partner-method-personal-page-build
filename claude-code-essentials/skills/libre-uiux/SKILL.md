# LibreUIUX — Synthesis Master

Orchestrate all dimensions of UI/UX into production-ready interfaces. Never ship without all seven pillars.

## The Seven Pillars

1. **Meaningful** — Does this serve users psychologically? Is the symbolism coherent?
2. **Beautiful** — Timeless principles applied. Would Dieter Rams see unnecessary elements?
3. **Accessible** — WCAG 2.1 AA minimum. Keyboard, screen reader, cognitive accessibility.
4. **Secure** — XSS/CSRF protected, input sanitized, CSP headers.
5. **Performant** — Core Web Vitals passing, lazy loading, bundle optimized.
6. **Tested** — Unit, accessibility, visual regression tests.
7. **Documented** — API docs, usage examples, design tokens extracted.

## Design Mastery Toolkit

### Visual Hierarchy
- Scale: Golden ratio (1:1.618) or major thirds (1.25, 1.5, 2, 3)
- F-pattern (reading) or Z-pattern (scanning)
- Strategic whitespace for emphasis

### Typography
- Pairing: Serif + sans, geometric + humanist
- Hierarchy: Display → Title → Body → Caption
- 4px/8px baseline grid rhythm
- 45-75 characters per line

### Color Theory (60-30-10 Rule)
- 60% dominant, 30% secondary, 10% accent
- Psychological: Blue=trust, red=urgency
- WCAG contrast ratios always

### Composition
- Rule of thirds, golden ratio proportions
- Swiss 12-column grid precision
- Negative space is not empty — it's intentional

### Gestalt Principles
- Proximity creates relationships
- Similarity groups elements
- Continuity guides the eye
- Figure/Ground separation

## Legendary Masters Reference
- **Vignelli**: Grid systems, typographic order
- **Dieter Rams**: Less but better, honest materials
- **Müller-Brockmann**: Mathematical composition
- **Paula Scher**: Bold typographic expression
- **Saul Bass**: Reductive symbolism
