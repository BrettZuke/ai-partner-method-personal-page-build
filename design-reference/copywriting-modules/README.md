# Copywriting Modules — Copy Elite First Principles

> Nine standalone modules from Copy Elite's "Copywriting 101 — First Principles" curriculum. Each is a self-contained framework you apply when writing any persuasive copy: hero headlines, FAQ answers, CTA copy, sales emails, DM scripts, video scripts.

The summary of these frameworks lives in [`../copywriting-frameworks.md`](../copywriting-frameworks.md). These are the **deeper, individual modules** — read them when you need to go beyond the summary into specific techniques.

---

## The 9 modules

| Module | Core idea |
|---|---|
| [`specific-conditions.md`](./specific-conditions.md) | Reader + Words = Action. Understand the lock before crafting the key. General conditions (all humans) + specific conditions (this audience). |
| [`text-subtext.md`](./text-subtext.md) | Every sentence carries a literal text + an implied subtext. Master the subtext layer to plant beliefs without stating them outright. |
| [`liking-relatability.md`](./liking-relatability.md) | Two of the strongest psychological levers. People buy from people they like and relate to. Specific tactics for engineering both. |
| [`pain-confidence.md`](./pain-confidence.md) | The buyer must feel pain (problem they want gone) AND confidence (you can solve it). Imbalance breaks the sale. |
| [`problem-chains.md`](./problem-chains.md) | Connecting surface problem → underlying problem → root problem → outcome. Most copy stops at surface; great copy goes to root. |
| [`urgency-scarcity.md`](./urgency-scarcity.md) | Real urgency vs. manufactured urgency. When each works, when each backfires. The ethics + mechanics of forcing decision. |
| [`concise-curiosity.md`](./concise-curiosity.md) | The trade-off between brevity and intrigue. Shorter ≠ better. Specific ≠ better. The two have to balance. |
| [`conditions-deck-template.md`](./conditions-deck-template.md) | The actual template for assembling a "conditions deck" before writing copy. Covers: psychographics, prior beliefs, objections, dream state, emotional state at point of read. |
| [`synthesis-checklist.md`](./synthesis-checklist.md) | Pre-publish checklist. Run every piece of finished copy through this before sending. Catches the most common mistakes. |

## How to use these

### When writing the website hero
1. Read [`specific-conditions.md`](./specific-conditions.md) to understand the reader
2. Read [`pain-confidence.md`](./pain-confidence.md) to balance the headline
3. Apply [`text-subtext.md`](./text-subtext.md) to imply rather than state
4. Run [`synthesis-checklist.md`](./synthesis-checklist.md) before publishing

### When writing FAQ answers
1. Read [`problem-chains.md`](./problem-chains.md) to find the real objection behind the question
2. Read [`text-subtext.md`](./text-subtext.md) to handle objections without naming them
3. Apply [`liking-relatability.md`](./liking-relatability.md) — answers should feel like a friend explaining

### When writing cold DMs / sales emails
1. Read [`specific-conditions.md`](./specific-conditions.md) — never DM without the dossier
2. Apply [`concise-curiosity.md`](./concise-curiosity.md) — sub-10 words on cold opens
3. Apply [`liking-relatability.md`](./liking-relatability.md) — match the audience's tone

### When writing the close / CTA
1. Apply [`urgency-scarcity.md`](./urgency-scarcity.md) — only if real
2. Apply [`pain-confidence.md`](./pain-confidence.md) — close needs both
3. Run [`synthesis-checklist.md`](./synthesis-checklist.md)

## Read order (if you have time to do them all)

If you're new to persuasive copy, read in this exact order:

1. `specific-conditions.md` (foundation)
2. `text-subtext.md` (the subtle layer that separates pros from amateurs)
3. `pain-confidence.md` (the emotional engine)
4. `problem-chains.md` (going from surface to root)
5. `liking-relatability.md` (the trust layer)
6. `urgency-scarcity.md` (the close mechanic)
7. `concise-curiosity.md` (the editing layer)
8. `conditions-deck-template.md` (apply it all to a real audience)
9. `synthesis-checklist.md` (final pass before any copy ships)

Total reading time: ~90 minutes. Apply once and you're better than 95% of copywriters working today.
